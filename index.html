Hello, I'm

Heejung Choi

Web Designer

Design with a Solid Foundation

이유 있는 디자인을 지향합니다.

디자인은 단순한 시각적 표현이 아니라, 사용자 경험과 목적을 기반으로 명확한 이유를 가져야 한다고 생각합니다.

사용자의 흐름을 고려한 UI를 설계하고, Figma와 Photoshop, Illustrator를 활용해 이를 구현할 수 있습니다.

Problem / Insight

더 나은 사용자 경험은 문제를 정확하게 정의하는 것에서 시작된다고 생각합니다. 사용자의 행동을 관찰하고, 그 안에서 숨겨진 불편의 이유를 발견합니다.

  • 사용자 중심 사고

    사용자의 입장에서 행동과 경험을 이해하고, 그 안에서 발생하는 불편과 니즈를 발견하며 더 나은 방향으로 개선하고자 합니다.

  • 문제 해결 중심 사고

    단순히 시각적인 결과물에 그치지 않고, 문제의 원인을 분석하고 논리적인 근거를 바탕으로 효과적인 해결 방안을 도출하는 데 집중합니다.

  • 관찰 기반 분석

    피사용자의 행동을 세밀하게 관찰하고, 그 안에서 문제의 원인을 파악하며 의미 있는 인사이트를 도출합니다.

Process

사용자의 경험을 설계하는 과정은 단순한 디자인을 넘어, 흐름을 구조화하는 일이라고 생각합니다. 리서치와 분석을 기반으로, 더 나은 방향을 끊임없이 고민하며 구체화합니다.

  • 반응형 웹 설계

    다양한 디바이스 환경에서도 일관된 경험을 제공할 수 있도록 유연한 레이아웃과 구조를 설계합니다.

  • 논리적 문제 해결

    문제의 원인을 구조적으로 분석하고, 명확한 근거를 바탕으로 최적의 해결 방안을 도출합니다

  • 경험 흐름 설계

    사용자의 행동과 맥락을 고려하여 자연스럽고 끊김 없는 경험 흐름을 설계합니다.

Skills

UX 기획부터 UI 디자인, 퍼블리싱까지 전 과정을 기반으로 일관된 사용자 경험을 구현할 수 있습니다. 각 단계의 흐름을 이해하고, 목적에 맞는 결과물로 연결할 수 있습니다.

  • Figma

    사용자 경험을 기반으로 UI 설계와 와이어프레임, 프로토타입 제작을 수행하고 컴포넌트를 활용해 협업 효율을 높일 수 있습니다.

  • Photoshop

    팝업, 배너, 상세페이지, 포스터 등 다양한 콘텐츠 제작과 이미지 보정 및 합성을 통해 완성도 높은 비주얼 디자인을 구현할 수 있습니다.

  • Illustrator

    아이콘, 로고 등 벡터 기반 그래픽 제작을 통해 브랜드 아이덴티티를 일관성 있게 시각적으로 표현할 수 있습니다.

  • html5

    웹 표준과 접근성을 고려한 시맨틱 마크업으로 검색 엔진에 최적화된 구조를 설계하고 사용자 중심의 정보 구조를 구성할 수 있습니다.

  • CSS3

    미디어 쿼리를 활용한 반응형 웹과 다양한 레이아웃을 구현하고 트랜지션 및 애니메이션을 통해 인터랙티브한 사용자 경험을 제공할 수 있습니다.

  • JavaScript

    Vanilla JS를 활용해 슬라이더, 모달, 스크롤 애니메이션 등 인터랙션을 구현하고 ES6+ 문법 기반의 가독성 높은 코드를 작성할 수 있습니다.

  • Git

    프로젝트 변경 사항을 버전별로 관리하고 브랜치 생성 및 병합을 통해 기능을 독립적으로 개발할 수 있습니다.

  • GitHub

    로컬 작업물을 원격 저장소에 공유해 협업할 수 있으며, GitHub Pages를 활용해 정적 웹 사이트를 배포하고 프로젝트를 외부에 공개할 수 있습니다.

  • Notion

    디자인 가이드라인과 회의 내용을 체계적으로 정리하고 문서화하여, 기록을 기반으로 한 명확한 커뮤니케이션과 효율적인 협업을 이끌 수 있습니다.

Popup Designs

Eye-Catching Design

나이키 팝업

하늘의 질감을 살린 푸른 배경에 투명한 로고 오브제를 배치하여 제품의 청량감과 세련된 무드를 극대화했습니다.

타이포그래피를 레이아웃 중심에 배치하고 하단에 유리 질감의 버튼을 더해 시각적인 주목도와 클릭률을 높인 팝업 디자인입니다.

떡볶이팝업

식욕을 자극하는 따뜻한 오렌지 톤 배경에 실제 메뉴 이미지를 우측에 크게 배치하여 시선을 압도하는 동시에 메뉴의 매력을 직관적으로 전달했습니다.

볼드한 타이포그래피와 노란색 뱃지 오브제를 활용해 혜택 정보를 명확히 강조하고, 손글씨 포인트를 더해 친근하게 디자인했습니다.

블랙프라이데이팝업

다크 모드 배경에 3D 캐릭터 텍스트와 핑크 컬러를 매치하여 자칫 무거울 수 있는 블랙 프라이데이 테마를 키치하고 위트 있게 재해석했습니다.

역동적인 'SALE' 띠 오브제와 중앙 집중형 레이아웃으로 시선의 흐름을 유도하고, 하단에 명확한 CTA 버튼을 배치하여 이벤트 참여 전환율을 높였습니다.

여행팝업

비행기와 여행 가방 오브제를 프레임 밖으로 과감하게 돌출시킨 레이아웃을 통해 팝업의 입체감과 생동감을 극대화했습니다.

좌측의 세로형 타이포그래피와 하단의 글로벌 네트워크 라인을 배치해 여행의 상징성을 높였으며, 차분한 민트 톤으로 신뢰감 있고 쾌적한 UX 디자인을 구현했습니다.

신학기팝업

새 학기의 설렘과 밝은 에너지를 전달하기 위해 파스텔 톤의 옐로우와 블루 컬러를 메인으로 사용하고, 볼륨감 있는 3D 텍스트 디자인으로 주목도를 높였습니다.

귀여운 아이와 강아지 일러스트를 중앙에 배치하고 주위에 꿀벌과 튤립 오브제를 더해, 타겟 고객인 어린이와 학부모의 시선을 사로잡는 친근하고 키치한 디자인을 완성했습니다.

크리스마스팝업

딥 그린 배경에 골드 라인의 트리 일러스트와 섬세한 빛망울 효과를 매치하여 연말 특유의 화려하고 고급스러운 무드를 시각화했습니다.

아치형 프레임 구조와 우아한 스크립트 폰트를 활용해 클래식한 감성을 더했으며, 하단에 텍스트 링크 형태의 CTA를 배치해 깔끔하고 완성도 높은 레이아웃을 구현했습니다.

빵팝업

브라운 컬러 톤과 밀 이삭 일러스트를 활용하여 갓 구운 빵의 풍미와 따뜻한 베이커리의 감성을 시각적으로 통일감 있게 구현했습니다.

곡선형 레이아웃 위에 제품 이미지를 배치해 부드러운 인상을 주었으며, 볼드한 타이포그래피와 필기체 폰트를 조합해 신메뉴 출시 소식을 명확하면서도 감각적으로 전달했습니다.
